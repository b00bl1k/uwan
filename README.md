# Uwan Micro LoRaWAN Stack

Simple and lightweight LoRaWAN messaging library in pure C.

## Features
- LoRaWAN Specification 1.0.3
- [x] ABP
- [ ] OTAA
- [x] Class A
- [ ] Class C
- [x] Uplink
- [ ] Downlink
- [ ] MAC commands
- [x] SX127x
- [ ] SX126x

## Benefits
- pure C -> awesome
- lightweight -> less bugs
- unit tests -> less bugs
- hardware absctraction -> run anywhere
